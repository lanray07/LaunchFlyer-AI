# LaunchFlyer AI

LaunchFlyer AI is a premium SwiftUI iOS app scaffold for generating complete promotional campaign packs from a prompt or voice input.

The app is mock-AI enabled by default and includes:

- SwiftUI NavigationStack + TabView app shell
- MVVM view models
- SwiftData persistence models
- StoreKit 2 subscription scaffolding
- Speech-to-text voice campaign builder
- Template, design, and export engines
- PNG, JPG, PDF export pipeline placeholder
- Native share sheet
- Brand kit and campaign library storage
- WidgetKit and Apple Watch architecture placeholders

Open `LaunchFlyerAI.xcodeproj` in Xcode, select the `LaunchFlyer AI` scheme, and run on an iOS 17+ simulator.
